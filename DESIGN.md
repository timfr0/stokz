# Stokz Design Reference

Source of truth: Google Stitch project `projects/3150647976352949774`

## Palette
- Background: `#f5f0e8`
- Primary: `#1a1a1a`
- Yellow accent: `#ffcc00`
- Red accent: `#e63b2e`
- Blue accent: `#0055ff`

## Typography
- Headlines and labels: Space Grotesk
- Body copy: Inter

## Surface rules
- Thick black borders
- Offset block shadows instead of soft drop shadows
- Flat fills
- Minimal radius
- Strong hierarchy with dense information blocks

## Responsive rules
- Desktop: watchlist scan first, analysis second
- Mobile: chart-first cards, four stats always visible, analysis stacked vertically
