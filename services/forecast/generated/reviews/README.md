# Stokz Daily Reviews

Each dated folder under this directory represents one full daily review package.

Expected files per day:
- review.json
- review.md
- tomorrow-config.json
- news-log.json
- market-context.json
- report-summary.md

The review package is intended to capture:
- what the model predicted
- what market context mattered
- what worked / what missed
- what tomorrow changes are proposed
- what the analyst role decided about those proposed changes

The analyst role exists to stop the system from making reckless threshold changes off weak evidence.
